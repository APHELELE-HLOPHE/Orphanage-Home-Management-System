# Orphanage Home System

A full-stack application to manage the orphanage home. Users can view children in the system, book their visit, make donations. The system consist of a **backend in Java** , **frontend in Flutter**, and **MySQL for database** 

## Features

### Frontend (Flutter)
- View available children
- Book a visit to the orphanage
- Make donations

### Backend (Java)
- REST API endpoints for frontend
- Database management using MySQL
- Handle donations, visits, and children records

## Prerequisites
- Java 17+
- Flutter SDK
- MySQL

### Backend setup
1. clone the project
2. open the project on eclipse
3. Configure the database: download the mysql.jar file from the official website and add it on the project
4. create the database named: OrphanageHome_db
5. run the backend project

### Frontend setup
1. open the frontend project on VS code
2. on the terminal, run the command: flutter run -d chrome
